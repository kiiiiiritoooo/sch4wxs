# 手势控制说明

## 功能概述

现在项目支持以下手势交互：

### 初始状态
- **默认显示蛋糕**：页面加载后直接显示3D蛋糕，无需等待启动页

### 左手手势

1. **左手五指张开** 👋
   - 触发：蛋糕散开
   - 效果：蛋糕粒子向四周散开，同时出现200颗粉色闪烁的星星
   - 状态：进入 `SCATTERED` 状态

2. **左手握拳** ✊
   - 触发：回归蛋糕形状
   - 效果：散开的粒子重新聚合成蛋糕形状
   - 状态：回到 `CAKE` 状态

### 右手手势

1. **右手握拳** ✊
   - 触发：在握拳位置显示照片
   - 效果：根据右手在屏幕上的位置，显示不同的照片
   - 位置区域：
     - 左上角 → 照片 0
     - 右上角 → 照片 1
     - 左下角 → 照片 2
     - 右下角 → 照片 3
     - 中间区域 → 照片 4
   - 移动：右手移动时，照片会跟随并切换到对应区域的照片

2. **右手张开五指** 👋
   - 触发：隐藏照片
   - 效果：当前显示的照片淡出隐藏

## 技术实现

### 手势识别
- 使用 MediaPipe Hands 识别左右手
- 通过计算指尖到手腕的距离判断手势状态
- 五指张开：平均距离 > 0.15
- 握拳：平均距离 < 0.08

### 照片配置
在代码中修改 `PHOTOS` 数组来配置不同位置的照片：

```javascript
const PHOTOS = [
    { src: './src/1.jpg', position: 'top-left' },      // 左上
    { src: './src/1.jpg', position: 'top-right' },     // 右上
    { src: './src/1.jpg', position: 'bottom-left' },  // 左下
    { src: './src/1.jpg', position: 'bottom-right' }, // 右下
    { src: './src/1.jpg', position: 'center' }         // 中间
];
```

### 星星效果
- 200颗粉色星星粒子
- 闪烁动画（亮度变化）
- 轻微浮动效果

## 使用建议

1. **准备照片**：建议准备5张不同的照片，分别对应5个区域
2. **手势识别**：确保光线充足，手部清晰可见
3. **摄像头权限**：首次使用需要允许摄像头权限
4. **浏览器兼容**：推荐使用 Chrome 或 Edge 最新版

## 注意事项

- 手势识别需要一定时间稳定，建议保持手势1-2秒
- 如果识别不准确，可以调整 `isHandOpen` 和 `isHandFist` 函数中的阈值
- 照片路径需要确保正确，建议使用相对路径
